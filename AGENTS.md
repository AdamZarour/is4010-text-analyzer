# AGENTS.md

I used AI as a helper while building this project, mostly for planning the file structure, generating a first pass of the text-analysis functions, and checking whether the README covered the rubric. It sped up the boring setup parts a lot.

It was not perfect. A couple times the first draft was a little too polished or a little too "framework-y" for a small class project, so I had to trim it back and make the code simpler. I also checked the logic myself, especially around sentence splitting and test cases, because the first version of any generated code is not something I would trust blindly.

The main thing I learned is that AI is useful when I already know what I want and can review the result critically. It saved time, but I still had to decide scope, fix mistakes, and make sure the final program actually matched the assignment.
